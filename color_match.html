Game Design

My initial focus was on the gameplay. After some brainstorming on how to incorporate colors into a game, I decided to present the user with a gradient of random colors and the task to find a certain color in a progressively shortening period of time. I started sketching how the game would be set up:

I decided to use the interaction on the screen as a component to increase difficulty. In the first, easiest mode of the game, a white bar contains a color and a slot in the middle so that itâ€™s easier for the player to find the correct color. This limited the difficulty by limiting the axis of change to the x-axis. In more challenging modes, two-dimensional gradients and a circle would cause the player to find the correct color in a larger, more complex gradient through moving the circle along both the x and y axis.

User Journey

With the goal of keeping the game simple and easy to use for casual games, I tried to keep the number of screens to switch between to a minimum. I created a simple flow between a start screen and the game, with the start screen containing a scroll wheel to select the game mode. Though slightly unconventional for a game, I decided that the scroll wheel was the easiest method for a user to select a game mode and start the game from one screen.

Outcome

As this project was started as more of a coding and development project, I began developing a working prototype in XCODE with Swift. At this time, I had not fully discovered my passion for User Experience/Product design, so I did not complete many stages of the product development cycle that I wish I would have, such as initial user feedback to confirm the game, presenting the prototype friends, families, and strangers to gain feedback and iterate on the design, or narrowing the niche of the target users, but I leave this project in my portfolio as it was the spark for my discovery into design.

I majored in Computer Science with the goal of becoming a front-end developer, but through researching this project, I discovered that most of what I have done on projects would be classified as user experience design and I began to hone-in my skills through taking the appropriate classes and additional research.
